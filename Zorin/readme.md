# Zorin
A Colorway based on ZorinOS's GTK theme, inspired by schnensch's [Zelk](https://betterdiscord.app/theme/Zelk) theme

### Importing:
```css
@import url(//dablulite.github.io/DiscordColorways/Zorin/import.css);
```
